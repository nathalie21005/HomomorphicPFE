
for show.py to work properly: 
1) the file must be put in the src folder 
2) the src folder is at the same level as the faces folder 
3) the face folder must contain: 
   a) rawdata folder (uncompressed)
   b) faceR 
   c) faceS 
4) in the src folder put the files: 
	a) eigenfaces.npy 
	b) meanface.py 
5) for this script to work, the following libraries must be installed: 
	a) numpy 
	b) matplotlib / pylab
